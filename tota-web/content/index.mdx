# tota documentation

**Soul-driven AI agent** — v0.0.4 · 60+ tools · 11 AI providers · WhatsApp · Browser · Computer-use · Android

Permission-hardened tools, token budgets, and multi-channel access.
Remembers what matters. Asks before it acts. Runs 24/7 from CLI, Telegram, WhatsApp, or REST API.

## What is tota?

tota is a self-hosted assistant that runs locally, connects to your preferred channels, and uses a rich toolset for coding, research, and automation. You stay in control of data, permissions, and where it runs.

## Quick start

### 1. Install

```bash
npx tota-agent
```

### 2. Run the setup wizard

```bash
tota
```

### 3. Start chatting

Ask it to do real work:

```text
Summarize the last 20 commits and open a PR if tests pass.
```

Need the full install guide? See [Getting Started](/docs/getting-started/installation).
